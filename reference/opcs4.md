# OPCS4 data

OPCS4 data

## Usage

``` r
data(opcs4)
```

## Format

Data frame with 9 columns

- code_1digit:

  The first digit of the OPCS4 code, or 'chapter'

- name_1digit:

  The name/group of 'chapter' of the OPCS4 code

- ...:

  Others to do with file is updated
