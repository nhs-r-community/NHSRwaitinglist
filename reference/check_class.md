# Check Class of Argument Inputs

Check Class of Argument Inputs

## Usage

``` r
check_class(..., .expected_class = "numeric", .call = rlang::caller_env())
```

## Arguments

- ...:

  Objects to be checked for class.

- .expected_class:

  Character. The name of the class against which objects should be
  checked.

- .call:

  The environment in which this function is to be called.
